## Run Local

Clone the project

```bash
  git clone https://github.com/MoisesGauna/moisesgaunaapp
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm create vite@latest
```

Start the server

```bash
  npm run dev
```
